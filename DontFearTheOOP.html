Predators are based on the idea of fish. They are identical, except that they have some additional qualities, namely sharp teeth, a certain species of fish, some level of salt in their veins, and a certain amount of fish eaten. Your standard predator will look mean, start the day out without any salt in their veins, and not yet eaten any fish. Whenever the predator gets hungry, the amount of fish eaten will go up by one. If someone asks a predator how much fish they have eaten, the predator will always respond with the amount of fish eaten. If the predator is not having a good time in the water, then add one to the predators salt level. Then print out "Oh no! (the specified predator) isn’t having a good time!" Predators extends the idea of Fish. Every predator has sharp teeth. Every predator has a species. Every predator will have some level of salt in their veins. Every predator will eat a certain amount of fish. For a given Predator, They will have sharp teeth. They will be a certain species/ They will start out with a salt level of 0. They will start out with no fish eaten. notHavingAGoodTime saltLevel increases by one howUpsetAmI tell them how salty they are eatAFish (name) add one to the number of fish this predator has eaten. print "Oh no! (the specified fish) has been eaten!"

public class Predators extends Fish {
boolean sharpTeeth;
String species;
int saltLevel;
int fishEaten;
Fish fish;
public Villains() {
sharpTeeth = true;
saltLevel = 0;
fishEaten = 0;
}
public void notHavingAGoodTime() {
saltLevel++;
}
public int howUpsetAmI() {
return saltLevel;
}
public void eatAFish (Fish fish) {
this.fish= fish;
fishEaten++;
System.out.println("The Predator has eaten " + fish.whatIsYourName());
}
}
