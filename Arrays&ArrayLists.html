Arrays & ArrayLists


Arrays and ArrayLists are two integral parts when it comes to programming. Arrays and ArrayLists are both used to store elements of the same type into a list. Arrays however, must have their size predetermined while ArrayLists do not. Arrays can either be instantiated with just a size or the actual elements. ArrayLists are only for objects and not primitive data types (unless you use the Integer, Double or Boolean class) while arrays can use any of them.

int[] test = new int[5]
int[] test2 = new int[]{1, 2, 3, 4, 5}

The code above shows an example of declaring an array with the size instantiated or the actual elements being instantiated.

Arrays have one built in method being the length method. The equivalent for ArrayLists is the size() method. Both these methods return the length of their respective lists. These lengths are generally used when you use a for loop to go through each item in the array or ArrayList. However, if you wish to just access each item in the array or ArrayList, you can simply use a for each loop. The for each loop traverses through each item in the array/ArrayList, but never actually edits the list.

The List interface is the interface that is implemented into ArrayLists. The List interface is an extension of the Collections interface and gives the default methods of void add(int index, Object E), Object remove(int index), Object get(int index), and Object set (int index, Object new). Due to List being an interface you cannot create a list object, but instead you are able to create a ArrayList object.
