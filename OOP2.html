OOP 2

Here you will learn about Polymorphism. This is an integral part of making an object oriented program.

Firstly, Polymorphism is the ability to change from an object made from a more broad class to an object from a more specific subclass of the superclass. A class is considered polymorphic when it has an is-a relationship with multiple other classes. An is-a relationship means that the class inherits something from another class, while a has-a relationship means that the class uses instance variables or references to objects from another class.
